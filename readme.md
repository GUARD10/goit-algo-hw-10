## Завдання 1. Оптимізація виробництва
- Початкові ресурси: вода 100, цукор 50, лимонний сік 30, фруктове пюре 40.
- Оптимальний випуск (PuLP): лимонад 30, фруктовий сік 20, разом 50 одиниць.
- Використані ресурси: вода 80, цукор 30, лимонний сік 30, фруктове пюре 40.
- Залишки після виробництва: вода 20, цукор 20, лимонний сік 0, фруктове пюре 0.
- Запуск демо:
  ```
  python task_1/task_1.py
  ```

## Завдання 2. Інтеграл методом Монте-Карло
- Функція `f(x) = x^2`, межі `[0, 2]`.
- Налаштування демо: 5 експериментів по 50_000 семплів (seed=0).
- Оцінка Монте-Карло: `2.665317445763128`.
- Аналітичне значення: `2.6666666666666665`.
- Перевірка SciPy `quad`: `2.666666666666667`, заявлена похибка `2.96e-14`.
- Похибка MC vs аналітика: `~0.00135` (~0.05%).
- Висновок: Monte Carlo дає близький результат з очікуваною стохастичною похибкою; збільшення семплів/експериментів зменшує різницю, що підтверджується збігом з аналітикою та `quad`.
- Запуск демо:
  ```
  python task_2/task_2.py
  ```
- Порівняння з `quad`:
  ```
  python -m task_2.quad_check
  ```
